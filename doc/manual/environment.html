Environment Variables

Using environment variables, you can configure some aspects of DTN2. The following table summarizes these variables.

Variable Component Syntax Description
DTNAPI_ADDR dtnd and all DTN API clients (dtncp, dtnping, etc) An IP address (not a hostname) This is the address that dtnd listens on for requests from DTN API clients, and the address that DTN API clients send requests to.
DTNAPI_PORT dtnd and all DTN API clients (dtncp, dtnping, etc) An IP port number This is the port that dtnd listens on for requests from DTN API clients, and the port that DTN API clients send requests to.
